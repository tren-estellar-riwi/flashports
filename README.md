# flashsports
Flashsports es la página predilecta para encontrar toda clase de eventos deportivos en tu zona y al rededor del mundo.  
¿Te gusta el Football y no sabes dónde hay eventos en los que participar? ¿No sabes dónde promocionar tu evento de skateboarding local?  
Flashsports es tu solución a ambos problemas, registrate ya y participa en toda clase de eventos deportivos, que van desde lo familiar hasta lo competitivo!
